3 **A contiguous growable generic type array in C programming language.**
4 
5 Vector supports amortized constant time insertion and removal of elements , as well as constant time access.

7 C doesnâ€™t allow data types to be passed as parameters. That means a generic container needs to manually manage memory in terms of the client element size, not client data type. You can do generic prommaming in c, without to use macros.
